# InventoryApp

InventoryApp is a console-based application for managing personal inventory. It allows users to create items, assign locations, and generate reports using a simple terminal interface.

## Quick Start

Clone the repository:

```
git clone https://github.com/spoonula/CS690-FinalProject.git
```

Navigate to the project:

```
cd CS690-FinalProject/InventoryApp
```

Run the application:

```
dotnet run
```

## Features

- Create and manage inventory items
- Assign items to locations
- Browse and view item details
- Generate inventory reports as CSV files
